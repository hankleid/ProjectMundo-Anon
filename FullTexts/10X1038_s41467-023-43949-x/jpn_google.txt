非線形検出器による高次元空間情報の量子輸送

要約: 離れた2者間で物理的に情報を転送せずに情報を共有する情報交換は、将来の量子ネットワークの重要なリソースです。高次元状態でこれを行うと、情報容量の向上とノイズ耐性の向上が期待されますが、これまでの進歩は限られています。ここでは、非線形パラメトリックプロセスが空間自由度で任意の高次元状態投影を可能にする方法を示します。強力なコヒーレントフィールドがプロセスの確率を高めます。これにより、単一のエンタングルメントペアと非線形空間モード検出器を備えた量子チャネルによって促進される高次元空間情報の量子転送を実験的に実現できます。和周波発生を使用して、エンタングルメントペアから光子の1つをアップコンバートし、高次元空間情報をもう1つに転送します。任意の光子空間モードに対してd = 15の量子チャネルを実現します。これは、送信される状態に関する知識を必要とせずに、軌道角運動量、エルミートガウス、および任意の空間モード重ね合わせにエンコードされた情報を忠実に転送することで実証します。私たちの実証は、構造化光の非線形制御という新興分野と量子プロセスを融合し、高次元の量子状態を利用するための新しいアプローチを提供するものであり、他の自由度にも拡張される可能性があります。

情報交換は現代社会のバックボーンであり、私たちの世界は光ファイバーと地上リンクのグローバル ネットワークで結ばれています。量子技術により、この交換は根本的に安全になり、初期の量子グローバル ネットワークの原動力となっています。たとえば、量子鍵配布では、ピア ツー ピア (通常はアリスとボブ) から鍵を交換して、通信当事者間で送信された情報を復号します。量子秘密共有では、そのような鍵を多数のノード間で分割します。量子セキュア直接通信では、鍵を使用せずに送信量子状態でエンコードします。これらすべての方式では、従来の方式と同様に、情報は送信者と受信者の間の物理リンクを介して送信されます。リモート状態準備では、リンクを介して物理的に情報を送信せずに当事者間で情報交換できますが、送信者 (アリス) は送信する情報を知っている必要があります。テレポーテーションでは、物理的なリンクを必要とせずに、遠く離れた当事者間で保護された情報交換を行うことができます。これは、もつれ光子の共有と従来の通信チャネルによって促進されますが、送信される情報はアリスに知られてはなりません。

前述の方式はすべて、高次元の量子状態を使用することで、より高いチャネル容量、セキュリティ、またはノイズ耐性を実現できるという利点があります。 光の空間モードを基盤とする文脈では、軌道角運動量 (OAM) は、パスやピクセルと同様に、高次元への潜在的なルートとして特に有用で話題であることが証明されています。 しかし、実験の進歩は遅く、光ファイバーでの鍵共有は最大 d = 6、自由空間での鍵共有は最大 d = 7、秘密共有は最大 d = 11 であることが示されています。 私たちの関心は、テレポーテーションなど、情報がリモートで共有され、物理的に送信されない方式にあります。テレポーテーションは、OAM を使用すると d = 2、パスの自由度を使用すると d = 3 に制限されています。 これまでのところ、これらのアプローチはすべて、状態制御と検出に線形光学を使用しており、高次元状態のコンテキストでは限界があることがわかっています。最近では、非線形光学が空間的に構造化された古典光の生成、制御、検出の刺激的なツールとして登場しましたが、偏光量子ビット測定を超えて空間的に構造化された量子状態を制御する方法は見つかっていません。高次元の量子情報処理と通信に非線形アプローチを使用する理論的スキームが提案されていますが、実験的に実証されたものはまだありません。

ここでは、量子チャネルを形成する 2 つのエンタングルされた光子と、情報エンコード用の明るいコヒーレント光源を使用して、任意の次元の非線形空間量子輸送方式を実験的に実証します。エンタングルされたペアからの光子の 1 つは、コヒーレント ビームを情報キャリアと効率向上の両方として使用して非線形結晶でアップコンバートされ、単一光子の検出に成功すると、2 光子同時測定によって他の光子に情報が輸送されます。私たちのシステムは、次元と基底に依存しない方法で空間情報を処理し、量子チャネルのモード容量は、ビーム サイズや結晶特性などのパラメーターによって簡単に制御できます。これらのパラメーターは理論的に概説され、d = 15 次元まで実験的に確認されています。光の空間モードをエンコード基底として使用し、このチャネルを使用して、OAM、エルミート ガウス分布、およびそれらの重ね合わせを含む多くの空間基底にわたって表現された情報を転送します。私たちの実験は完全な理論的処理によってサポートされており、非線形光学制御と検出によって高次元構造化量子状態を利用するための新しいアプローチを提供します。

私たちのコンセプトの概略図を図 [1] に、実験的実現を図 [2] a に示します。詳細は補足ノート [1] に記載されています。2 つのエンタングルメント光子 B と C は、共線非縮退自発パラメトリックダウンコンバージョン (SPDC) 用に構成された非線形結晶 (NLC1) から生成されます。光子 C は、空間光変調器 (SLM A) を使用して準備された転送対象状態 (コヒーレントソース A) と相互作用するために送られ、光子 B は空間光変調器 (SLM B) と SMF による空間投影によって測定されます。

私たちの方式では、コヒーレント ソース A からの光子を、第 2 の非線形結晶 (NLC2) 内の単一光子 C と重ね合わせ、和周波発生 (SFG) によって生成されたアップコンバージョンされた光子 D を検出します。このプロセスが成功するには、エンタングルメント ペアからの単一光子 B と同時に単一光子 D (エンタングルメント ペアからの単一光子 C による) が測定される必要があります。アップコンバージョンの確率を高めるために、コヒーレント状態を入力として使用します。この場合、すべての光子は、転送したい同じモード情報を運びます。

このプロセスをよりよく理解するには、OAM モードを例に挙げると有益です。完全な基底に依存しない理論的処理は、補足ノート [2] から 4 に記載されています。SPDC 結晶を、それぞれ方位角インデックス ℓ p= 0 と半径インデックス p p= 0 のラゲール ガウス モードでポンプします。OAM は SPDC プロセスで保存されるため、ℓ p= 0 =ℓ B+ℓ C となります。アップコンバージョン プロセスでも OAM が保存されるため、ℓ D= 0 の空間モードのみをサポートするシングル モード ファイバー (SMF) によって検出が行われる場合、ℓ D= 0 =ℓ A+ℓ C となります。A と B の両方が C と共役である場合、ℓ A=ℓ B= −ℓ C の場合にのみ一致が検出され、準備された状態 (A) が転送された状態 (B) と一致することがすぐにわかります。より一般的には（補足ノート [2] を参照）、光子 D の検出が最初の SPDC ポンプと同じモードになるように設定されている場合（光子 D を反ポンプと呼ぶ）、アップコンバージョン プロセスは SPDC プロセスの共役として機能し、アップコンバージョンに関与するコヒーレント ソース A の各光子の状態は光子 B の状態に移されることを示すことができます。言語を明確に保つために、アップコンバージョンに関与するコヒーレント ソース A の光子を光子状態 A と呼びます。これは、ダウンコンバージョン プロセスに関与するポンプ光子が 1 つだけであると見なされる SPDC プロセスの場合と同様であり、両方のケースで真空項は無視されます。これは、プロセスで一致が生じないためです。しかし、アップコンバージョンを利用した量子輸送は、適切な実験条件、すなわち、選択された基底におけるSPDCプロセスからの信号光子とアイドラー光子間の完全な反相関と、光子状態Aと光子C間の反相関を保証するために長さと位相が一致するアップコンバージョン結晶の下でのみ発生します（詳細については補足ノート[3]を参照）。

どちらの場合も、測定された状態 (実線バー) は準備された状態 (透明バー) と非常によく一致しています。この結果では、入力状態の密度行列の対角項が正常に転送されたことのみが確認され、量子転送の前後のコヒーレンス (密度行列の非対角要素) の転送は確認できないことに注意してください。初期状態と最終状態の対角要素がよく一致していることは、これらの要素に対して量子転送が機能している証拠であり、d = 4 までの完全な位相情報と d = 15 までのチャネル容量 (位相を含む) によって裏付けられています。最終状態の d = 9 の対角項を定量化するために、量子状態トモグラフィーから忠実度を決定するのに時間がかかりすぎるため (カウントが少ないため)、類似性を尺度として使用しますが (方法を参照)、この尺度では準備された状態と測定された状態のモード位相が考慮されないことに注意してください。転送された状態の例の最終的な要約は、図[6]に示されており、2次元から9次元まで、多くの基底にわたっています。準備された状態（透明なバー）と転送された状態（実線のバー）は、類似性から判断するとよく一致しており、チャネルの品質を確認しています。検出されたOAM状態（実線のバー）の一致カウントが示されていることに注意してください。準備されたもの（透明なバー）の重み付けは、視覚化のために正規化された確率を示すことを目的としています。

構造化量子光は最近注目を集めており、情報処理と通信のためのより大きなヒルベルト空間を約束しています。高次元量子状態の作成における非線形光学の使用は網羅的です (SPDC、フォトニック結晶、共鳴メタサーフェスなど)。一方、非線形プロセスでのエンタングルメントとコヒーレンスの保存は、効率的な光子検出、特に通信波長の光子の測定に使用されています。しかし、非線形プロセスによる高次元量子状態を完全に利用および制御することは、依然として困難です。注目すべき例外には、時間周波数領域での進歩、高次元状態を利用するもう 1 つの自由度 (時間モードの効率的なデマルチプレックスとトモグラフィー測定のための量子パルス ゲートの実証など)、差周波数生成による多重化の逆プロセス、スペクトル的に区別可能なソースの量子干渉、高次元情報のエンコード、量子波動関数の同時時間成形と検出などがあります。私たちの知る限り、私たちの研究は空間領域における初の研究であり、非線形プロセスによる空間量子情報の制御と処理のための刺激的なリソースを提供します。高次元のスペクトル時間状態制御とオンチップ非線形ソリューションの進歩を空間自由度と組み合わせることで、量子ビットを超えた量子情報処理の新たな展望が開ける可能性があります。

結論として、我々は非線形検出器を使用して未知の状態の投影を実行するためのエレガントな方法を示し、高次元および多くの空間基底にわたる量子情報を、量子リソースとして 1 つのエンタングルメント ペアのみを使用して転送できるようにしました。我々の結果は、ノイズ抑制やバックグラウンド減算なしでチャネルの非古典的な性質を検証しました。我々の量子輸送スキームは、転送する状態を多数のコピーでエンコードする必要があるためエンタングルメントをテレポートすることはできませんが、それでもレーザー光子の状態を遠くにある以前にエンタングルメントされた光子に安全に転送し、レーザー光子の状態に関する知識を使用せずにこれを行います (輸送からテレポートへの移行の課題については、補足ノート [10] および [15] を参照してください)。重要なことは、我々の包括的な理論的処理により、SPDC 結晶および検出器でのモード サイズなど、量子輸送チャネルのモード容量を決定する調整可能なパラメーターが概説され、わずかな実験的調整 (たとえば、レンズの焦点距離) のみが必要であることです。私たちのチャネルのモード容量は実験リソースによってのみ制限されますが、将来の研究では、関連パラメータの選択を最適化し、非線形プロセスを改善することで、転送モードの数を増やすことを目指すことができます。私たちの研究は、このアプローチが未知の高次元空間状態の量子輸送に抱く刺激的な展望を強調しており、将来的には、多自由度と高次元量子制御のためのハイブリッドエンタングルメント（偏光と空間）とハイパーエンタングルメント（空間と時間）状態などの混合自由度に拡張される可能性があります。
