利用非线性探测器进行高维空间信息的量子传输

摘要：两个远距离方之间的信息交换是未来量子网络中的关键资源，即信息共享但不进行物理传输。利用高维状态实现信息交换有望提高信息容量并增强抗噪声能力，但迄今为止的进展有限。在这里，我们展示了非线性参数过程如何允许在空间自由度中进行任意高维状态投影，其中强相干场增强了该过程的概率。这使我们能够通过实验实现高维空间信息的量子传输，该传输由具有单个纠缠对和非线性空间模式检测器的量子通道实现。使用和频生成，我们将纠缠对中的一个光子上变频，从而将高维空间信息传输到另一个光子。我们实现了任意光子空间模式的 ad = 15 量子通道，我们通过忠实地传输编码为轨道角动量、厄米-高斯和任意空间模式叠加的信息来证明这一点，而无需了解要发送的状态。我们的演示将结构光非线性控制的新兴领域与量子过程相结合，为利用高维量子态提供了一种新方法，并且也可以扩展到其他自由度。

信息交换是现代社会的支柱，我们的世界通过全球光纤和地面链路网络连接起来。量子技术使这种交换从根本上变得安全，推动了新生的量子全球网络的发展。例如，量子密钥分发在对等方（通常是 Alice 和 Bob）之间交换密钥以解码通信双方传输的信息，量子秘密共享将这样的密钥拆分给许多节点，量子安全直接通信则不需要密钥发送密钥，而是以传输的量子态进行编码。在所有这些方案中，就像其经典对应方案一样，信息通过发送方和接收方之间的物理链路发送。远程状态准备允许各方之间交换信息，而无需通过链路物理传输信息，但发送者（Alice）必须知道要发送的信息。隐形传态，允许远距离各方之间进行受保护的信息交换，而无需物理链路，通过共享纠缠光子和经典通信信道来实现，其中发送的信息不能被 Alice 知道。

所有上述方案都将受益于使用高维量子态，提供更高的通道容量、安全性或抗噪声能力。在以光的空间模式为基础的背景下，轨道角动量（OAM）已被证明特别有用和具有现实意义，–，路径和像素也是如此，作为通向高维的潜在路线。然而，实验进展缓慢，在光纤中显示的共享密钥最多为d = 6，在自由空间中为d = 7，共享秘密最多为d = 11。我们感兴趣的是信息远程共享而不是物理发送的方案，例如隐形传态，使用OAM时将其限制为d = 2，使用路径自由度时则限制为d = 3。到目前为止，所有这些方法都使用线性光学进行状态控制和检测，但这在高维状态的背景下存在已知的局限性。最近，非线性光学已成为一种令人兴奋的空间结构化经典光的创造、控制和检测工具，但除了偏振量子比特测量之外，它还没有找到控制空间结构化量子态的方法。尽管已经提出了使用非线性方法进行高维量子信息处理和通信的理论方案，但尚未通过实验证明。

在这里，我们通过实验演示了任意维度的非线性空间量子传输方案，使用两个纠缠光子形成量子通道，并使用明亮的相干源进行信息编码。纠缠对中的一个光子在非线性晶体中上转换，使用相干光束作为信息载体和效率增强器，成功进行单光子检测，通过双光子巧合测量将信息传输到另一个光子。我们的系统以与维度和基础无关的方式处理空间信息，我们的量子通道的模式容量很容易通过光束大小和晶体特性等参数控制，我们在理论上概述了这些参数，并通过实验证实了高达 d = 15 个维度。使用光的空间模式作为我们的编码基础，我们使用此通道传输跨许多空间基础表达的信息，包括 OAM、Hermite-Gaussian 及其叠加。我们的实验得到了完整的理论处理支持，并提供了一种通过非线性光学控制和检测来利用高维结构化量子态的新方法。

在我们的方案中，我们将来自相干源 A 的光子与第二个非线性晶体 (NLC2) 中的单个光子 C 重叠，并检测通过和频生成 (SFG) 产生的上转换光子 D。该过程的成功取决于对单个光子 D（来自纠缠对的单个光子 C）与来自纠缠对的单个光子 B 的测量是否一致。我们使用相干状态作为输入来增强上转换的概率，其中所有光子都携带我们想要传输的相同模式信息。

为了更好地理解该过程，以 OAM 模式为例将大有裨益；补充说明 [2] 至 4 中给出了完整的与基础无关的理论处理。我们用方位角和径向指数分别为ℓ p= 0 和p p= 0 的拉盖尔-高斯模式泵浦 SPDC 晶体。OAM 在 SPDC 过程中得以守恒，因此ℓ p= 0 =ℓ B+ℓ C。上转换过程也守恒 OAM ，因此如果检测是由仅支持ℓ D= 0 的空间模式的单模光纤 (SMF) 进行的，则ℓ D= 0 =ℓ A+ℓ C。可以立即看到，只有当 A 和 B 都与 C 共轭时才会检测到巧合，ℓ A=ℓ B= −ℓ C，因此制备状态 (A) 与传输状态 (B) 匹配。可以更一般地证明（见补充说明 [2]），如果将光子 D 的检测配置为与初始 SPDC 泵浦相同的模式（我们可以称光子 D 为反泵浦），则上转换过程充当 SPDC 过程的共轭，并且相干源 A 中参与上转换的每个光子的状态都被传输到光子 B 的状态。为了保持语言清晰，我们将把相干源 A 中参与上转换的光子称为光子状态 A ，就像在 SPDC 过程中一样，其中只有一个泵浦光子被视为参与下转换过程，在这两种情况下都忽略真空项，因为它们不会引起我们过程中的巧合。然而，上转换辅助量子传输仅在相关的实验条件下才能发生，即在所选基础上来自 SPDC 过程的信号光子和闲置光子之间存在完美的反相关性，以及上转换晶体的长度和相位匹配以确保光子态 A 和光子 C 之间存在反相关性（有关详细信息，请参阅补充说明 [3]）。

结构化量子光近来受到广泛关注，有望为信息处理和通信提供更大的希尔伯特空间。非线性光学在高维量子态创建中的应用非常广泛（SPDC、光子晶体、共振超表面等），而非线性过程中纠缠和相干性的保存已使其用于高效光子检测，特别是用于测量电信波长光子。然而，通过非线性过程完全利用和控制高维量子态仍然难以实现。值得注意的例外包括在时频域方面取得的进展，这是利用高维状态的另一个自由度，例如量子脉冲门的演示，用于有效解复用时间模式以及断层扫描测量，通过差频生成进行复用的逆过程，光谱可区分源的量子干涉，高维信息编码以及同时进行时间整形和检测量子波函数。据我们所知，我们的工作是空间领域的首例，为通过非线性过程控制和处理空间量子信息提供了令人兴奋的资源。将高维光谱时间状态控制和片上非线性解决方案的进展与空间自由度相结合，可能预示着量子信息处理在量子比特之外的新前景。

总之，我们展示了一种使用非线性探测器对未知状态进行投影的优雅方法，从而促进了高维度和跨多个空间基的量子信息传输，只需一对纠缠对作为量子资源即可。我们的结果验证了信道的非经典性质，无需任何噪声抑制或背景减法。虽然我们的量子传输方案无法传送纠缠，因为需要对要传输的状态进行多份编码，但它仍然可以安全地将激光光子的状态传输到遥远的、之前纠缠的光子，而且它不需要了解激光光子的状态（参见补充说明[10]和[15]，讨论了从传输到传送的挑战）。重要的是，我们全面的理论处理概述了决定量子传输通道模式容量的可调参数，例如 SPDC 晶体和探测器的模式尺寸，只需要进行微小的实验调整（例如，透镜的焦距）。我们通道的模式容量仅受实验资源的限制，而未来的研究可以通过优化相关参数的选择和改进的非线性过程来增加传输模式的数量。我们的工作凸显了这种方法对未知高维空间状态的量子传输的激动人心的前景，并且将来可以扩展到混合自由度，例如混合纠缠（极化和空间）和超纠缠（空间和时间）状态，以实现多自由度和高维量子控制。
